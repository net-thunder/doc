# 客户端启动

## Windows、Macos启动

- 进入首页
![](/resource/client.png)


- 配置app/application.yaml文件

![](/resource/client-setting.png)

```yaml
# 租户code
tenantId: default
# api服务地址
httpServer: 127.0.0.1:11805
```

**提示：MacOS通过右键显示包内容进入编辑**
![](/resource/macos-show-content.png)

- 点击启动，连接服务器成功
![](/resource/client-connect.png)
